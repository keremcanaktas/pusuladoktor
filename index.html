# Pusula Privacy Policy


Last updated: May 11, 2026


This Privacy Policy explains how the Pusula mobile application collects, uses, and protects information while providing its services.


## 1. Information We May Access


The Pusula application may request access to the following information or device capabilities in order to provide its features:


- Camera access

- Data transmitted over the internet for application functionality

- Information or content provided by the user within the application


## 2. Camera Permission


The application requests camera access only to enable in-app camera usage or scanning features such as barcode or QR-related workflows.


Camera data:


- is used only when initiated by the user

- is not automatically transferred outside the device unless required for the requested feature

- is processed only for application functionality


## 3. How Information Is Used


Collected or processed information may be used to:


- operate application features

- improve user experience

- detect technical or performance issues

- provide services securely and accurately


## 4. Data Sharing


User data is not shared with unauthorized third parties except where required by law or necessary for the technical delivery of the service.


## 5. Data Security


Pusula takes reasonable technical and administrative measures to protect data against unauthorized access, alteration, disclosure, or loss.


## 6. User Rights


Users may request information, correction, or deletion regarding their data, subject to applicable law.
